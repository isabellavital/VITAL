# VITAL - Elegant Community Platform

Welcome to **VITAL** - an elegant, interactive community platform built with modern web technologies. VITAL provides a seamless experience for users to stay connected through announcements, events, schedules, and community resources.

## 🎯 Project Overview

VITAL is a full-stack web application designed to be user-friendly, visually elegant, and highly interactive. It features an intuitive admin dashboard where administrators can manage announcements, host events, schedule meetings, and communicate with other admins through a dedicated chat room.

## ✨ Features

### Public Features
- **🏠 Home Page** - Display announcements and featured content
- **📋 About Page** - Learn about VITAL and the community
- **📝 Application Page** - Submit applications to join or participate
- **📊 Survey Page** - Participate in community surveys
- **📚 Resources Page** - Access helpful materials and links

### Scheduling & Events
- **📅 Schedule Tab** - View all upcoming events and meetings
- **📍 Event Details** - Descriptions, dates, times, and locations
- **✅ Interest Marking** - Mark interest in events to track attendance
- **👥 Attendance Dashboard** - See how many people are interested

### Admin Features
- **🔐 Secure Admin Login** - Role-based authentication
- **📢 Announcement Management** - Post and manage announcements
- **🎪 Event Hosting** - Create and schedule events
- **💬 Admin Chat Room** - Real-time communication between admins
- **⚙️ Admin Dashboard** - Complete management panel

### Design
- **🎨 Green Theme** - Elegant, nature-inspired color scheme
- **📱 Responsive Design** - Works on desktop, tablet, and mobile
- **⚡ Interactive UI** - Smooth animations and user interactions
- **♿ User-Friendly** - Intuitive navigation and clear information hierarchy

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI library
- **Tailwind CSS** - Styling with green theme
- **React Router** - Navigation
- **Axios** - HTTP client
- **Socket.io Client** - Real-time chat
- **React Icons** - Icon library
- **Framer Motion** - Animations

### Backend
- **Node.js** - Runtime
- **Express.js** - Web framework
- **MongoDB** - Database
- **JWT** - Authentication
- **Socket.io** - Real-time communication
- **Multer** - File uploads
- **Bcryptjs** - Password hashing

## 🚀 Installation

### Prerequisites
- Node.js (v16 or higher)
- MongoDB (local or Atlas)
- npm or yarn

### Backend Setup

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file and configure your environment variables

4. Start the backend server:
```bash
npm run dev
```

### Frontend Setup

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```
